# PRD: 新增中文 README 文档

## 背景
当前仓库已有英文 `README.md`，用户要求添加一份中文文档的 README。

## 目标
在仓库根目录提供中文版本的 `README.md`，保留与当前项目使用方式相关的核心信息，提升中文读者可读性。

## 范围
- 修改文件：`README.md`
- 以中文为主重新组织说明
- 保留关键内容：项目简介、架构、快速开始、核心命令/注意事项

## 非目标
- 不改动业务代码
- 不新增功能实现
- 不引入与 README 无关的结构调整

## 验收标准
- `README.md` 主要内容为中文
- 文档结构清晰，可用于新用户快速上手
- 术语与命令示例与当前仓库保持一致
